# **MITgcm Code Configuration**

## **Overview**

code files for the configuration

---

## **Contents**

- **CPP_OPTIONS.h**  
  model flags.

- **DIAGNOSTICS_SIZE.h**  
  size and number of diagnostic outputs

- **OBCS_OPTIONS.h**  
  open boundary condition options

- **SIZE.h**  
  grid dimensions 

- **SIZE.h_no_mpi**  
  grid dimensions for non-MPI

- **packages.conf**  
  active model packages
